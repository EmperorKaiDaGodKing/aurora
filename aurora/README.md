# Project Aurora — underwear for moods

A mood-driven underwear that listens. Aurora translates how you feel into curated underwear suggestions and small self-care rituals — quiet, tactile, and a little dangerous.

## Why Aurora
Underwear hold memory. Aurora turns ambient inputs (weather, calendar, mood, listening history) into underwear and rituals that help you show up intentionally.

## Features
- Mood → Underwear mapping with weight for comfort / boldness
- Daily ritual suggestions (music, scent, micro-ritual)
- Saved looks, tags, and adaptive recommendations
- Lightweight client (React/Next) with a small, composable design system

## Tech stack
- Frontend: React + Next.js, Tailwind CSS
- Data: local-first, optional sync, simple JSON store
- Design: Figma (design tokens & motion)
- Deploy: Vercel (preview + production)

## Getting started
1. git clone https://github.com/EmperorKaiDaGodKing/aurora
2. cd aurora
3. pnpm install
4. pnpm dev
Open http://localhost:3000
